# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-03-25

### Added
- Initial release of Sapo CLI tool
- Command to download and install JFrog Artifactory OSS
- Version listing with size and timestamp information
- Support for multiple platforms (Windows, macOS, Linux)
- Progress bars for downloads and installations
- Checksum verification for downloaded packages
- Automatic cleanup of temporary files
- Rich CLI output with colored status indicators
- Version sorting with latest versions displayed first
- Package size formatting in human-readable format

### Development
- Comprehensive test suite with 90% code coverage
- Poetry-based dependency management
- GitHub Actions CI workflow for testing and building
- Cross-platform testing on Windows, macOS, and Linux
- Documentation in README.md with installation and usage instructions
- Type hints throughout the codebase
- Linting with ruff
- Security checks with bandit and safety
